# CH-N 4105 Considerations

> This document is **not** legal or regulatory advice.  
> It describes design intentions to behave in a grid-code–friendly manner.

## Design principles

- Avoid sustained export above configured thresholds.
- React quickly to changes in household load and PV production.
- Prefer conservative behaviour when measurements are uncertain or stale.
- Provide clear configuration for maximum allowed export per phase.

## Implementation notes

- Export limits are computed per phase based on live measurements.
- When data is missing or invalid, SmartFlow can:
  - Fall back to safe default limits, or
  - Disable export (configurable).

You must verify compliance with your DSO and local regulations.
